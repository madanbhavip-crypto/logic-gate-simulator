# 🔌 Logic Gate Simulator

An interactive simulator for basic logic gates (AND, OR, NOT, XOR) — pick a gate, enter inputs, and see the output plus the full truth table.

Built with HTML, CSS, and JavaScript — no frameworks, no hardware, just code.

## How it works
1. Choose a gate from the dropdown
2. Enter Input A (and B, if needed)
3. Click "Get Output" to see the result and the full truth table
